# Real-Time-Multi-Camera-Tracking-System-proj

Built a real-time multi-camera tracking system processing 10+ FPS streams with cross-camera identity matching, reducing ID-switch errors by 35%

+ Object detection (YOLOv8 / Detectron2)
+ Re-identification (ReID models)
+ Multi-camera ID consistency
+ Stream processing (Kafka / WebSockets)

Tech Stack:
+ PyTorch, OpenCV
+ FastAPI + Redis
+ Docker
