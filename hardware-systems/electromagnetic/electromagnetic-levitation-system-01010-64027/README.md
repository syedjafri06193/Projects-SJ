# Electromagnetic Levitation System

An advanced Computer Engineering + Physics project that uses **electromagnetic fields** to levitate and stabilize a small metallic object in mid-air using a real-time feedback control system.

This project combines:

- Electromagnetism
- Embedded systems
- Real-time control
- Sensor integration
- Control theory
- Power electronics

The system dynamically adjusts coil current through a feedback loop to counteract gravity and maintain stable levitation.

---

# Demo Concept

The electromagnet generates a magnetic field strong enough to attract a ferromagnetic object upward.

The challenge is stability.

Without feedback control, the object either:
- crashes into the electromagnet
- or falls due to insufficient magnetic force

A PID-controlled real-time system continuously reads sensor data and adjusts the magnetic field strength to keep the object suspended.

---

# Physics Behind the Project

## Lorentz Force

The levitating force comes from electromagnetic interactions.

```math
\vec{F}=q(\vec{E}+\vec{v}\times\vec{B})
```

In practical electromagnets, magnetic force generated by the coil depends on:
- current through the coil
- magnetic field intensity
- distance from the object

The electromagnet must generate enough upward force to counter gravity:

```math
F_{magnetic}=mg
```

# PID Controller

The stabilization loop is the most important part of the project.

```math
u(t)=K_p e(t)+K_i\int e(t)dt+K_d\frac{de(t)}{dt}
```

Where:
- \(K_p\) = proportional gain
- \(K_i\) = integral gain
- \(K_d\) = derivative gain
- \(e(t)\) = position error

The PID controller constantly corrects oscillations and keeps the object stable.
