# Physics Behind the System

## Faraday’s Law of Electromagnetic Induction

A changing magnetic field induces voltage in a conductor.

```math
\mathcal{E} = -\frac{d\Phi_B}{dt}
```

Where:
- \(\mathcal{E}\) = induced electromotive force
- \(\Phi_B\) = magnetic flux

The rapidly changing magnetic field generated by the induction coil induces circulating currents inside the metal.

---

# Eddy Currents

The induced currents inside the conductive material are called eddy currents.

These currents experience electrical resistance and generate heat:

```math
P = I^2R
```

Where:
- \(P\) = thermal power
- \(I\) = induced current
- \(R\) = electrical resistance

The metal effectively becomes its own heating element.

---

# Magnetic Field Generation

The induction coil produces an alternating magnetic field.

```math
B = \mu n I
```

Where:
- \(B\) = magnetic field strength
- \(\mu\) = magnetic permeability
- \(n\) = turns per unit length
- \(I\) = current

Higher-frequency currents increase induction efficiency.

# Resonant Circuit Design

Most induction heaters use an LC resonant tank circuit.

```math
f = \frac{1}{2\pi\sqrt{LC}}
```

Where:
- \(f\) = resonant frequency
- \(L\) = inductance
- \(C\) = capacitance

Resonance dramatically improves efficiency and power transfer.

# Closed-Loop Temperature Control

To maintain stable heating, the system continuously adjusts power output using a PID controller.

```math
u(t)=K_p e(t)+K_i\int e(t)dt+K_d\frac{de(t)}{dt}
```

Where:
- \(e(t)\) = temperature error
- \(u(t)\) = control signal

The controller prevents:
- overheating
- thermal runaway
- unstable temperature oscillations
